# vuex-count

## Project setup
```
yarn
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```

### 版本
vue 2.6.x
vue-cli 3.x

### 案例优化
+ element 组件按需加载
+ 路由懒加载（两种方案）

### vuex（两种方案）
+ this.$store
+ 导入import {} from 'vuex'
